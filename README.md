# ROS-with-React-Web-App-via-rosbridge-and-websockets
ROS1 TurtleBot3 Burger simulation connected to a React web app via rosbridge and WebSockets. The browser handles RRT exploration, TF transforms, path following (P-controller), teleop (WASD), SLAM reset, and E-stop, creating a full Explore → Follow loop directly from a zero-install web interface.
